Операции с числами

  •

Взаимодействовать с числами в JavaScript можно по разному. Например, иногда их используют в качестве констант и выводят на экран. А иногда применяют к ним математические операции.

Об этих операциях мы сейчас и поговорим.

Базовые операции

С числами в JavaScript можно производить все математические операции, которым нас учили в школе: сложение ( + ) , вычитание ( - ), умножение ( * ), деление ( / ) и возведение в степень ( ** ).

let sum = 1 + 2; // 3

let division = 4 / 2; // 2

let multiply = 2 * 2; // 4

let subtraction = 5 - 3; // 2

let exponentiation = 5 ** 2 // 25

Также имеются сокращённые варианты данных операций. Такой вариант позволяет сразу обновить значение переменной после операции:

let example = 1; // определяем через let, потому что значение переменной будет изменяться.

example += 1 // 2

example -= 1 // 1

example *= 2 // 2

example /= 2 // 1

example **= 2 // 1

А теперь сравните предыдущую запись с несокращённым вариантом:

let example = 1; // определяем через let, потому что значение переменной будет изменяться.

example = example + 1 // 2

example = example - 1 // 1

example = example * 2 // 2

example = example / 2 // 1

example = example ** 2 // 1

Остаток от деления

Также есть операция, которая называется «остаток от деления». Она записывается знаком процента ( % ).

Ее можно описать выражением «сколько останется, если одно число поделить на другое». Например: 9 % 2 = 1.

Чтобы это понять, мы должны задать себе вопрос.

Какое самое ближайшее к 9 число, которое делится на 2 без остатка, т.е. чётное?

8.

Сколько будет 9 - 8?

1.

Вот так и получаем, что остаток от деления 9 на 2 будет 1.

В реальных задачах часто применяют операцию определения остатка от деления. Так хотят узнать, является число чётным или нет.

let even = 8 % 2; // 0 - число 8 чётное

let odd = 11 % 2; // 1 - число 11 нечётное

Порядок операций

Если в выражении есть несколько операторов, то порядок их выполнения определяется приоритетом. Или, другими словами, существует определённый порядок выполнения операторов.

Из школы мы знаем, что умножение в выражении 1 + 2 * 2 выполняется раньше сложения. Это как раз и есть «приоритет». Говорят, что умножение имеет более высокий приоритет, чем сложение.

Скобки важнее, чем приоритет. Так что, если мы не удовлетворены порядком по умолчанию, мы можем использовать их, чтобы поменять приоритет. Например, написать (1 + 2) * 2.

В JavaScript много операторов. Каждый оператор имеет соответствующий номер приоритета. Тот, у кого это число больше — выполнится раньше. Если приоритет одинаковый, то порядок выполнения — слева направо.

Вот полная таблица приоритета операторов.

let firstExample = 1 + 2 * 3 // 7, потому что у умножения приоритет выше, чем у сложения

let secondExample = (1 + 2) * 3 // 9, потому что в выражении появились скобки

Работа с переменными

Производить вычисления мы можем не только с обычными числами, но и с переменными имеющими числовые значения.

Так например присвоив переменной first значение 3, а переменной second значение 5, мы можем сложить эти две переменные. Полученное значение мы присвоим переменной sum:

const first = 3;

const second = 5;

const sum = first + second; // 8

Также можно переприсвоить значение переменной, если она объявлена с помощью ключевого слова let:

let first = 5;

let second = 3;

let actionWithVariables = first + second; // 8

second = 4;

actionWithVariables = second - first; // -1
